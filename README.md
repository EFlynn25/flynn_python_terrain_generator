# flynn_python_terrain_generator
A simple terrain generator using matplotlib

I moved on to other things before I tried figuring out how to export this in another format or 3d object, so this is what I have for now.

**To use it: `python3 .\world_creator.py`**

## Options

`-s [seed]` : Set generation seed (default: 12345)

`-w [world_size]` : Set world width/height in chunks (default: 3)

`-z [size]` : Set chunk size (default: 16)

`-f [hill_frequency]` : Change frequency of hills (default: 3)

`-g [hill_height]` : Change height of hills (default: 5)

(and as always, use `-h` or `--help` to get this printed in the console)
